# Business Performance & Customer Insights

## Project Overview
This project analyzes business performance and customer behavior using Python, SQL, Excel, and Power BI.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Excel
- Power BI

## Project Structure
- data: Raw and cleaned datasets
- python: Data cleaning and analysis scripts
- sql: SQL queries and database analysis
- power_bi: Dashboards and visualizations

## Key Analysis
- Sales performance by region
- Monthly sales trends
- Top products by profit
- Customer purchase behavior

## Outcome
This project demonstrates end-to-end data analysis skills suitable for a Junior Data Analyst role.
